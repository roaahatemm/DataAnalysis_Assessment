# E-commerce Data Analysis Project

## Project Description
This project involves analyzing a self-created e-commerce dataset.  
It includes Python scripts for data generation and analysis, SQL scripts for querying and business insights, Power BI dashboards for visualization, and a full project documentation with insights and recommendations.  

The main goal is to demonstrate **data analysis skills**, **SQL querying**, **dashboard creation**, and **market basket analysis** for an e-commerce scenario.

---

## Folder Structure

- `Dataset/` – Contains the CSV files:
  - `customers.csv`
  - `orders.csv`
  - `order_items.csv`
  - `products.csv`
- `python_codes_for_ecommerce/` – Python scripts:
  - `DatasetCreation.py` – Generates the dataset
  - `MarketBasketAnalysis.py` – Performs market basket analysis using the Apriori algorithm
- `SQL/` – SQL scripts and documentation:
  - `EcommerceAssessment.sql` – Creates tables and answers business questions
  - `QueryOptimizationExplanation.pdf` – Explains query optimization techniques
- `Dashboard/` – Power BI dashboard:
  - `EcommerceDashboard.pbix`
  - Screenshots of dashboard pages
- `Ecommerce_Documentation/` – Project documentation PDF:
  - `EcommerceProjectDocumentation.pdf`
